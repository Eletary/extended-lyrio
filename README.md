# extended-lyrio

A Tampermonkey userscript that enhances the Lyrio OJ experience with modern UI improvements and productivity tools.

[![GitHub release](https://img.shields.io/github/v/release/Eletary/extended-lyrio)](https://github.com/Eletary/extended-lyrio/releases)
[![GitHub stars](https://img.shields.io/github/stars/Eletary/extended-lyrio)](https://github.com/Eletary/extended-lyrio/stargazers)

## ✨ Features

- [x] **Hitokoto** — Inspirational quotes on the sidebar, refresh anytime
- [x] **Typo Fixer** — Auto-fixes typos in page title and header (e.g., "nflsoj" → "NFLSOJ")
- [x] **Copy Code** — One-click code copying with visual feedback

Stay tuned for more features! 🚀

## 📦 Installation

1. Install [Tampermonkey](https://www.tampermonkey.net) for your browser.
2. Click the link below to install the script:
   👉 **[Install extended-lyrio](https://github.com/Eletary/extended-lyrio/releases/latest/download/extended-lyrio.user.js)**
